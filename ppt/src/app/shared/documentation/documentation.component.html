Welcome to the PPT Builder Documentation

Overview

PPT Builder application allows users to access the slides with Keydetails from the application and create their new presentation. Users can also upload their own presentations to split presentations into multiple slides, add Keydetails for each slide, and manage the slides through an interactive interface.

Features

  • Split presentations into multiple files
  • View and navigate through the split slides using a carousel
  • Add Keydetails (title, keywords, notes, rating) for each slide
  • Save Keydetails to the backend and handle errors
  • Display Toastr notifications for user feedback

Usage

Navigating the Application:

Uploading a Presentation

  • Click the Split PPT to add a new file and its associated keydetails.
  • Ensure to provide keydetails for each new file added.
  • Note that slides without the keydetails will not be stored in the application.
  • Now each separate slide will be saved in the application.

Adding Metadata

  • Fill in the form on the right side of the screen with the following fields:
    • Title: Enter a title for the slide.
    • Keywords: Enter keywords related to the slide.
    • Notes: Enter any notes you want to associate with the slide.
    • Rating: Select a rating (1-5 stars) for the slide based on the efforts you made to create slides.
    • Date: View the date associated with the slide.
  • Click "Save" to store the metadata.
  • Navigating Slides
    • Use the "Previous" and "Next" buttons to move between slides.
    • If the metadata is incomplete, a warning message will appear.

Search Slides

  • Search with the keywords in the search bar and hit search.
  • All the slides having the keyword you entered will appear.
  • Select the slides which you need to have in your new presentation.
  • Click the Download PPT button.
  • Enter the title and the purpose of the presentation and submit.
  • Now you have the newly prepared presentation.

Sign-Up Flow

  • Enter Email for Verification
    • The user navigates to the sign-up page.
    • The user is prompted to enter their email address.
    • Only email addresses from Ecanarys domains are allowed.
  • Submit Email
    • The user submits their email address.
    • An OTP is generated and sent to the provided email address.
    • The user is directed to the email verification page.
  • Verify Email with OTP
    • The user receives an email with an OTP.
    • The user enters the OTP on the verification page.
    • The application verifies the OTP.
    • If the OTP is valid, the user is redirected to the sign-up page.
    • If the OTP is invalid, an error message is displayed.
  • Complete Sign-Up
    • The user fills in the following details on the sign-up page:
      • First Name
      • Last Name
      • Password (with validation for strength and confirmation)
    • The user submits the sign-up form.
    • The application creates a new account with the provided details.
    • Upon successful account creation, the user is redirected to the login page.
  • Login
    • The user enters their email address and password on the login page.
    • The application verifies the credentials.
    • If valid, the user is logged in and redirected to the application dashboard.
    • If invalid, an error message is displayed.

Forgot Password Flow

  • Request Password Reset
    • Verify the email as described in the sign-up process.
  • Reset Password
    • Enter the new Password (with validation for strength and confirmation).
    • Submit the Password Reset Form.
    • Login with your new Password.

Error Handling

  • If metadata is not provided before navigating away from a slide, a warning will be shown.
  • Ensure that all required fields are filled else the slide will not be saved in the application.

Toastr Notifications

  • Success: Notifications will appear when keydetails are saved successfully.
  • Error: Notifications will appear for validation errors or any issues encountered.
  • Warning: Notifications will appear if keydetails are missing or incomplete.

Troubleshooting

  • Keydetails Not Saving: Ensure all required fields are filled before saving. Check for validation errors.
  • Slides Not Updating: Refresh the page whenever the searched slide is not getting displayed.
